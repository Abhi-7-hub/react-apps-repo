## ðŸš€ How to Run Locally

1. Clone the repo:

```bash
git clone https://github.com/Abhi-7-hub/react-apps-repo.git
cd React_apps/portfolio-react
```

2. Install dependencies:

```bash

```

3. Start development server:

